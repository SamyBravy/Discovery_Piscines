## Overview

This repository contains the exercises from the **Discovery Piscine Python** at 42. It is an introductory course designed to teach the fundamentals of Python programming, from basic syntax to object-oriented concepts.

The curriculum is structured into "cells", each focusing on a specific topic.

-   First scripts (`Hello World`).
-   Basic variables.

### [Cell 01 - Variables](cell01)
**Data Types & Operations**
-   String manipulation.
-   Integer and Float operations.
-   Input/Output.

### [Cell 02 - Conditions](cell02)
**Control Flow**
-   `if`, `elif`, `else` statements.
-   Logical operators.

### [Cell 03 - Loops](cell03)
**Iteration**

### [Cell 05 - Data Structures](cell05)

### [Cell 06 - Classes](cell06)
